# 微信黑科技项目文档

## 项目简介
微信黑科技项目基于微信Windows和安卓协议开发，提供了一套强大的微信API和Hook工具。我们的目标是通过兼容微信协议，为用户提供更开放和可定制的微信接入能力。

### 特色功能
- **原生接口**：对微信原生接口的直接封装。
- **二开接口**：基于原生接口的扩展，实现更多功能。
- **多设备支持**：单设备多开微信。
- **API目录**：[点击查看](menu.md)

### 使用必要条件
- **设备要求**：
  - 实体windows电脑win7以上系统
  - 云服务器windows电脑win7以上系统
- **微信版本要求**：
  - 3.9.5.*
  - 3.9.2.*
