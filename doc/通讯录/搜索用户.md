以下是搜索用户的信息以及请求和返回示例的Markdown格式：

## 搜索用户

### 简要描述

微信号、手机号、QQ号查询用户信息。

### 请求URL

```
http://127.0.0.1:8000/api/
```

### 请求方式

POST

### 参数

| 参数名    | 必选 | 类型   | 说明         |
| --------- | ---- | ------ | ------------ |
| type      | 是   | int    | 接口编号     |
| keyword   | 是   | string | 搜索关键词   |

### 请求示例

```json
{
  "type": 43,
  "keyword": "abcdefg"
}
```

### 返回示例

```json
{
  "error_code": 10000,
  "description": "",
  "data": {
    "status": 0,
    "desc": "",
    "data": []
  }
}
```

请注意，实际情况可能会根据您的应用程序的要求有所不同。返回的数据结构可能包括搜索到的用户信息。
