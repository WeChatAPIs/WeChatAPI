## 保存/取消保存群聊到通讯录

### 简要描述

将群聊保存或取消保存到通讯录。

### 请求URL

```
http://127.0.0.1:8000/api/
```

### 请求方式

POST

### 参数

| 参数名           | 必选 | 类型   | 说明               |
| ---------------- | ---- | ------ | ------------------ |
| type             | 是   | int    | 接口编号           |
| chatroomUserName | 是   | string | 群聊id             |
| isSave           | 是   | bool   | 是否保存           |

### 请求示例

保存群聊到通讯录：

```json
{
  "type": 10027,
  "chatroomUserName": "xxxx@chatroom",
  "isSave": true
}
```

取消保存群聊到通讯录：

```json
{
  "type": 10027,
  "chatroomUserName": "xxxx@chatroom",
  "isSave": false
}
```

### 返回示例

```json
{
  "error_code": 10000,
  "description": "",
  "data": {
    "status": 0,
    "desc": ""
  }
}
```

请注意，实际情况可能会根据您的应用程序的要求有所不同。
