# 发送表情信息

##### 简要描述

- 发送表情消息

##### 请求URL

- `http://127.0.0.1:8000/api/`

##### 请求方式

- POST

##### 参数

| 参数名       | 必选 | 类型     | 说明      |
|-----------|----|--------|---------|
| type      | 是  | int    | 接口编号    |
| userName  | 是  | string | 接收人wxid |
| emojiPath | 是  | string | 文件绝对路径  |

##### 请求示例

```json 
 {
  "type": 10011,
  "userName": "filehelper",
  "emojiPath": "C:\\xxx.gif"
}

```

##### 返回示例

```json
  {
  "error_code": 10000,
  "description": "",
  "data": {
    "status": 0,
    "desc": ""
  }
}

```
