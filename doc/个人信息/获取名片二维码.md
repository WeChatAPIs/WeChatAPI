# 获取名片二维码

## 获取个人名片二维码

### 请求URL

- `POST` http://127.0.0.1:8000/api/

### 参数

| 参数名  | 必选 | 类型  | 说明   |
|------|----|-----|------|
| type | 是  | int | 接口编号 |

### 请求示例

```json
 {
  "type": 14
}
```

### 返回示例

```
  {
    "error_code": 10000,
    "description": "",
    "data": {
      "status": 0,
      "desc": "",
      "data": []
    }
  }

```
