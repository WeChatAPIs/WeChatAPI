# 修改地区

## 简要描述

用于修改地区。

### 请求URL

- `POST` http://127.0.0.1:8000/api/

### 参数

| 参数名    | 必选 | 类型   | 说明     |
| --------- | ---- | ------ | -------- |
| type      | 是   | int    | 接口编号 |
| nation    | 是   | string | 国家     |
| province  | 否   | string | 省份     |
| city      | 否   | string | 市区     |

### 请求示例

```json
 {
  "type": 10050,
  "nation": "CN",
  "province": "Shanghai"
}

```
### 返回示例

```json
{
  "error_code": 10000,
  "description": "",
  "data": {
    "status": 0,
    "desc": ""
  }
}

```
