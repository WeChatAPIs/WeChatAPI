# 修改签名

## 简要描述

修改个性签名

### 请求URL

- `POST` http://127.0.0.1:8000/api/

## 参数

| 参数名     | 必选 | 类型   | 说明       |
| ---------- | ---- | ------ | ---------- |
| type       | 是   | int    | 接口编号   |
| signature  | 是   | string | 新签名     |

## 请求示例

```json
{
  "type": 10049,
  "signature": "1111111"
}
```


### 返回示例
```
{
  "error_code": 10000,
  "description": "",
  "data": {
    "status": 0,
    "desc": ""
  }
}
```
