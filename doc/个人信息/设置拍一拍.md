# 设置拍一拍

可以设置别人拍一拍自己时显示的内容。

## 请求URL

http://127.0.0.1:8000/api/

## 请求方式

POST

## 参数

| 参数名     | 必选 | 类型     | 说明    |
|---------|----|--------|-------|
| type    | 是  | int    | 接口编号  |
| content | 是  | string | 拍一拍内容 |

## 请求示例

```json 
{
  "type": 10047,
  "content": "并说你真帅"
}
```
## 返回
```json
  {
  "error_code": 10000,
  "description": "",
  "data": {
    "status": 0,
    "desc": ""
  }
}
```
