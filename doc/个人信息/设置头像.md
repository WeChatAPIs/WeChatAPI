# 设置头像

## 简要描述

修改头像

### 请求URL

- `POST` http://127.0.0.1:8000/api/

## 参数

| 参数名      | 必选 | 类型     | 说明     |
|----------|----|--------|--------|
| type     | 是  | int    | 接口编号   |
| filePath | 是  | string | 文件绝对路径 |

### 请求示例

```json
 {
  "type": 64,
  "filePath": "C:\\xxx.png"
}
```

### 返回示例

```json
  {
  "error_code": 10000,
  "description": "",
  "data": {
    "status": 0,
    "desc": ""
  }
}
```
