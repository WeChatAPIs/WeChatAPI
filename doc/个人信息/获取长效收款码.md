# 获取长效收款码

## 简要描述

获取长效收款码。

### 请求URL

- `POST` http://127.0.0.1:8000/api/

### 参数

| 参数名    | 必选 | 类型     | 说明       |
|--------|----|--------|----------|
| type   | 是  | int    | 接口编号     |
| amount | 否  | int | 金额（单位：分） |

### 请求示例

```json
 {
  "type": 10087,
  "amount": 100
}



```

### 返回示例

```json
  {
  "error_code": 10000,
  "description": "",
  "data": {
    "status": 0,
    "desc": "",
    "data": []
  }
}

```
