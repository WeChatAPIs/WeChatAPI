## 前置准备
```
具体安装步骤
1. 安装指定版本微信（提供的微信安装包是官方版本，如不放心可以自行搜索该版本的包）
2. 安装【微软常用运行库合集】必备的windows依赖
3. 打开【启动器 wechatAPI_xxxxxx.exe 】输入账号密码注册账号、注册后点击登录按钮进入启动界面
4. 点击【启动微信】扫码登录即可（只有该方式启动的微信才有api功能）
  1. 这时候就已经安装完了
5. 使用postman、apifox等软件按照文档调用接口就好
  1. cmd接口测试命令如下，复制命令打开CMD窗口执行即可

curl --location --request POST "http://127.0.0.1:8888/api/" --header "Content-Type: application/json" --data-raw "{  \"type\": 28 }"
```
