## 上传朋友圈图片

### 简要描述

上传朋友圈图片。

### 请求URL

```
http://127.0.0.1:8000/api/
```

### 请求方式

POST

### 参数

| 参数名   | 必选 | 类型   | 说明               |
| -------- | ---- | ------ | ------------------ |
| type     | 是   | int    | 接口编号           |
| filePath | 是   | string | 文件绝对路径       |

### 请求示例

```json
{
  "type": 5,
  "filePath": "C:\\xxx.jpg"
}
```

### 返回示例

```json
{
  "error_code": 10000,
  "description": "",
  "data": {
    "status": 0,
    "desc": ""
  }
}
```

这个接口用于上传朋友圈图片，您需要提供图片的绝对路径作为 `filePath` 参数。返回的数据包括了 `error_code`、`description` 以及 `data` 字段，您可以根据 `error_code` 来判断操作是否成功。
