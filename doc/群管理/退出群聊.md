## 退出群聊

### 简要描述

退出指定群聊。

### 请求URL

```
http://127.0.0.1:8000/api/
```

### 请求方式

POST

### 参数

| 参数名           | 必选 | 类型   | 说明         |
| ---------------- | ---- | ------ | ------------ |
| type             | 是   | int    | 接口编号     |
| chatroomUserName | 是   | string | 群聊wxid     |

### 请求示例

```json
{
  "type": 10028,
  "chatroomUserName": "xxx@chatroom"
}
```

### 返回示例

```json
{
  "error_code": 10000,
  "description": "",
  "data": {
    "status": 0,
    "desc": ""
  }
}
```

这个接口用于退出指定的群聊，返回的数据包括了 `error_code`、`description` 以及 `data` 字段，您可以根据 `error_code` 来判断操作是否成功。
