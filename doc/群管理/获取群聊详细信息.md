## 获取群聊详细信息

### 简要描述

获取指定群聊的详细信息。

### 请求URL

```
http://127.0.0.1:8000/api/
```

### 请求方式

POST

### 参数

| 参数名           | 必选 | 类型   | 说明     |
| ---------------- | ---- | ------ | -------- |
| type             | 是   | int    | 接口编号 |
| chatroomUserName | 是   | string | 群聊wxid |

### 请求示例

```json
{
  "type": 30,
  "chatroomUserName": "xxx@chatroom"
}
```

### 返回示例

```json
{
  "error_code": 10000,
  "description": "",
  "data": {
    "status": 0,
    "desc": "",
    "data": []
  }
}
```

请注意，实际情况可能会根据您的应用程序的要求有所不同。返回的数据结构可能包括了 `status` 和 `desc`，以及包含群聊详细信息的 `data` 列表。您可以根据实际需要进行处理和解析。
