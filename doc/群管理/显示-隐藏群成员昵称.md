## 显示/隐藏群成员昵称

### 简要描述

显示或隐藏群成员昵称。

### 请求URL

```
http://127.0.0.1:8000/api/
```

### 请求方式

POST

### 参数

| 参数名           | 必选 | 类型   | 说明               |
| ---------------- | ---- | ------ | ------------------ |
| type             | 是   | int    | 接口编号           |
| chatroomUserName | 是   | string | 群聊wxid           |
| isShow           | 是   | bool   | 是否显示           |

### 请求示例

```json
{
  "type": 10025,
  "chatroomUserName": "xxx@chatroom",
  "isShow": true
}
```

### 返回示例

```json
{
  "error_code": 10000,
  "description": "",
  "data": {
    "status": 0,
    "desc": ""
  }
}
```

请注意，实际情况可能会根据您的应用程序的要求有所不同。返回的数据结构可能包括了 `status` 和 `desc`，您可以根据实际需要进行处理和解析。
